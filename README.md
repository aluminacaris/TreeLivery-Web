# TreeLivery (starter)

This repository contains a minimal starter project: a FastAPI backend and a React + Tailwind frontend scaffold.

Features implemented so far:

User and restaurant registration and login
Home screen displaying restaurants
In-restaurant screen showing dishes
Incomplete ordering system
Restaurant administration screen, with complete crud options for dishes

Upcoming functions:

Finalize order system, back and front end.
Automatic recommendation system depending on user profile, with different types of diets and restrictions (possibly implementing a future AI...).
UI refinement.
